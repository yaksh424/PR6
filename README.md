# Демонстрация шаблонов в FastAPI — ПЗ№6

Простое учебное приложение, показывающее работу с Jinja2-шаблонами в FastAPI.

Ключевые моменты реализованы:
- Наследование шаблонов (base.html)
- Циклы и условные блоки в шаблонах
- Форма POST для создания записи
- Статические файлы (CSS)

Персонализация: В подвале страницы ненавязчиво указаны: Юдин Владислав Александрович, ИВТ-432Б.

Как запустить локально (macOS / zsh):

1) Создайте виртуальное окружение (рекомендуется):

```bash
source .venv/bin/activate
```

2) Установите зависимости:

```bash
pip install -r requirements.txt
```

3) Запустите приложение:

```bash
uvicorn main:app --reload
```

Откройте http://127.0.0.1:8000/ в браузере.

Примечание: Это учебный пример с in-memory-хранилищем; данные не сохраняются после перезапуска приложения.
